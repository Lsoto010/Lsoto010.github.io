LAB 1

This is one of the first labs I did. I enjoyed this lab because it introduced me to much coding, especially how to use the terminal and clone repositories to GitHub. My favorite part about the lab was learning together with my partner because, at first, I thought it would be solo.

LAB 7

In Lab 7, Part 2, I specifically enjoyed this lab because it was about blackjack, and I always wanted to learn it. This was more complicated than the others, but it taught me Looping. The lab required you to handle hands with any number of cards, which involved using loops to iterate over the given cards and perform calculations.

LAB 9

With Lab 9, part 2 was my favorite because it taught me the use of the program, not showing the number having off by implementing the secret and game programs; I gained practical experience in file I/O operations and learned how to separate responsibilities between different programs. It also helped me Perform error handling for file operations, such as checking if a file exists or if the data in the file is valid.

My thoughts on this class

Overall, these labs provided me with foundational knowledge in using the terminal, version control with Git and GitHub, loop constructs, file I/O operations, error handling, and collaboration. These skills are essential for further development as a programmer and will be beneficial in future projects and learning experiences.
